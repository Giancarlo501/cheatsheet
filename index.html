Positioning of HTML Elements

Property Description
Relative An element’s position is relative to its default position on the page.
Absolute An element’s position is relative to its closest positioned parent element
Sticky An element will stick to a defined offset position when the user scrolls its parent container
Z-Index This element specifies how far back or how far forward an element appears on the page when it overlaps other elements
Float This property can move elements as far left or as far right as possible on a web page
Clear You can clear an element’s left or right side (or both) using the clear property
